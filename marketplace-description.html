This plugin introduces new Intellij Shortcuts and aims to speed up development of tests:
  • Add Mockito Verify
  • Add Mock Method
  • Introduce Mocks To Field
  • Introduce Mocks To Variable

Add Verify Method (CTRL + M, CTRL + C)

Before

var string[cursor] = Mockito.mock(String.class);

After

var string = Mockito.mock(String.class);
Mockito.verify(string.compareTo([cursor]any());
Variable finder: You can also just put your cursor anywhere and the plugin will apply the shortcut to a selected variable

Add Mock Method (CTRL + M, CTRL + M)

Before

var string[cursor] = Mockito.mock(String.class);

After

var string = Mockito.mock(String.class);
Mockito.when(string.compareTo(any())).thenReturn([cursor]null);
Variable finder: You can also just put your cursor anywhere and the plugin will apply the shortcut to a selected variable

Introduce mock field (CTRL + M, CTRL + F)

Before

class IntroduceMockFieldExample {
    void example() {
        var result = new TestClass([cursor]null,null);
    }
}

After

class IntroduceMockFieldExample {
    @Mock
    String string1;
    @Mock
    String string2;

    void example() {
        var result = new TestClass(string1, string2);
    }
}

Introduce Mock To Variable (CTRL + M, CTRL + V)

Before

void example() {
    var result = new TestClass([cursor]null, null);
}

After

void example() {
    var string1 = Mockito.mock(String.class);
    var string2 = Mockito.mock(String.class);
    var result = new TestClass(string1, string2);
}

Multiline Count (CTRL + M, CTRL + 1)

A popup will to specify the start value and the interval between each steps. You can also insert characters to count from A to Z or z to a etc.

Before

void example() {
    var result1 = new TestClass([cursor]);
    var result2 = new TestClass([cursor]);
    var result3 = new TestClass([cursor]);
    var result4 = new TestClass([cursor]);
    var result5 = new TestClass([cursor]);
    var result6 = new TestClass([cursor]);
    var result7 = new TestClass([cursor]);
}

After

void example() {
    var result1 = new TestClass("1");
    var result2 = new TestClass("2");
    var result3 = new TestClass("3");
    var result4 = new TestClass("4");
    var result5 = new TestClass("5");
    var result6 = new TestClass("6");
    var result7 = new TestClass("7");
}

NamingSchemeCycling (CTRL + ALT + SHIFT + U)

Switch between different naming schemes. Select any text (with multi cursor support!), press the shortcut to cycle between dot.case, camelCase, PascalCase, doener-case and SNAKE_CASE.

Before

    thisIsACoolTestInCamelCase

After

    THIS_IS_A_COOL_TEST_IN_CAMEL_CASE

Switch Mockito Mocking Variant (CTRL + M, CTRL + S)

Put cursor (multi cursors!) on a line and switch between Mockito.when and Mockito.doReturn mock method variants

Before

    (cursor)Mockito.doReturn("abcd").when(mockName).getStuff("test", "test", "test");
    (cursor)Mockito.when(mockName.getStuff("Test2")).thenReturn("abcd");

After

    Mockito.when(mockName.getStuff("test", "test", "test")).thenReturn("abcd");
    Mockito.doReturn("abcd").when(mockName).getStuff("Test2");
